## 📖 项目简介

这是一个专为 **Minecraft 服务器管理员 (Server Admin)** 打造的现代化插件索引平台。我们深知在海量资源中寻找合适插件的痛苦，因此通过极致的**像素风设计**与**智能筛选系统**，将 Spigot, Paper, Purpur 等核心的优质插件直观地呈现在你面前。

> [!TIP]
> **为什么选择本项目？** 纯前端实现，极致流畅，支持暗色模式，且完全开源。

---

## ✨ 核心特性

| 功能 | 描述 |
| :--- | :--- |
| 🎨 **像素美学** | 采用独特的 Minecraft 风格撞色设计，兼顾视觉冲击力与易用性 |
| 🔍 **精准检索** | 支持关键词即时搜索，配合左侧分类栏，秒级定位目标插件 |
| 📊 **深度透视** | 实时显示下载量、评分、付费状态及支持的服务端版本 (1.8 - 1.21+) |
| 🌓  **双色模式** | 完美适配亮色与暗色主题，保护深夜调服的肝 |
| 📱 **响应式布局** | 无论是在 PC 还是手机端，都能获得一致的浏览体验 |
| ⚡ **零后端开销** | 基于静态数据驱动，极速加载，不依赖服务器数据库 |

## 🛠️ 技术实现

项目坚持“轻量化”原则，使用最纯粹的技术栈构建：

* **核心引擎**: `Vanilla JavaScript (ES6+)` - 无框架负担，原生速度
* **样式构建**: `CSS3 (Flex/Grid)` - 现代布局方案
* **图标组件**: `FontAwesome 6.0`
* **字体支持**: `Google Fonts`
* **托管方案**: `Netlify / Vercel`

## 🚀 快速开始

### 1. 本地预览
如果你想在本地运行或修改该项目：
```bash
# 克隆仓库
git clone [https://github.com/Kauo7420/Kauo7420.git](https://github.com/Kauo7420/Kauo7420.git)

# 进入目录
cd Kauo7420

# 使用 Live Server 或任意静态服务器运行 index.html

```

### 2. 添加你的插件

插件数据存储在 `data.json` (或相应 JS 文件) 中，你可以通过提交 **Pull Request** 来贡献新插件。
